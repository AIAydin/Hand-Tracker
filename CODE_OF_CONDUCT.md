# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

- Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- FAQ: https://www.contributor-covenant.org/faq
- Translations: https://www.contributor-covenant.org/translations

Enforcement
- Report unacceptable behavior via GitHub (Issues or Discussions) or by contacting the maintainers.
- For security-related concerns, please follow the process in SECURITY.md (do not post sensitive details publicly).

This Code of Conduct applies within all project spaces and when an individual is officially representing the project in public spaces.
